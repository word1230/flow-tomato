# Flow Tomato - 心流番茄钟

一个极简、快速的Web端番茄钟应用，帮助用户将复杂任务拆解、进入心流状态、战胜拖延。

## 功能特点

- 🚀 **无登录系统**：通过自定义密钥快速进入个人空间，数据存储于本地
- 📝 **任务管理**：支持添加、完成、更新和删除任务
- 💡 **思路规划区**：每个任务关联一个文本编辑区，用于记录任务拆解思路
- 🍅 **番茄数预估**：为每个任务设定预估需要的番茄钟数量
- ⏰ **番茄钟计时器**：遵循标准的番茄工作法循环（25分钟工作/5分钟休息）
- 🔔 **智能提醒**：通过浏览器桌面通知和预设声音进行提醒
- ✅ **灵活的任务完成机制**：支持自动和手动标记任务完成
- 📊 **番茄统计**：实时显示累计完成的番茄总数
- 📱 **响应式设计**：适配桌面和移动设备

## 使用方法

1. 打开应用，输入任意字符串作为你的个人空间密钥
2. 添加任务，为每个任务设定预估的番茄钟数量
3. 在思路规划区记录任务拆解的思路和步骤
4. 选择任务，点击"开始"按钮启动番茄钟
5. 专注工作25分钟，然后休息5分钟
6. 每完成4个番茄钟，享受15分钟的长休息
7. 当任务完成或达到预估番茄数时，标记任务为完成

## 设计理念

Flow Tomato 遵循以下设计原则：

- **极简主义**：界面简洁，无干扰元素
- **快速响应**：所有交互即时反馈
- **专注导向**：帮助用户进入心流状态
- **数据隐私**：所有数据存储在用户本地，不上传到服务器
